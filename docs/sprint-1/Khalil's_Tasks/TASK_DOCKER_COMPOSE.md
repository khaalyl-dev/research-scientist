# Task: Docker Compose Setup

## 1. Task Overview

### Task Name
Docker Compose Configuration for Application Services

### Objective

Configure Docker Compose to automate the creation, configuration, and execution of project services.

Docker Compose allows managing multiple containers as a single application stack.

The objective of this task is to create a reproducible development environment containing the application components.

---

# 2. Docker Introduction

## Description

Docker is a containerization platform that packages applications with all their dependencies.

A Docker container contains:

- Application code
- Runtime environment
- Libraries
- System dependencies
- Configuration files

Benefits:

- Same environment for all developers
- Easy deployment
- Isolation between services
- Faster setup
