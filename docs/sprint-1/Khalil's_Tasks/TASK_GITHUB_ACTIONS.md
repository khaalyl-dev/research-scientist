# Task: GitHub Actions CI/CD Setup

## 1. Task Overview

### Task Name

Continuous Integration and Deployment using GitHub Actions

### Objective

Configure an automated workflow to verify project quality after every modification pushed to GitHub.

GitHub Actions automatically executes tasks such as:

- Installing dependencies
- Running tests
- Checking code quality
- Building applications
- Deploying applications

---

# 2. GitHub Actions Introduction

## Description

GitHub Actions is a CI/CD automation platform integrated into GitHub.

It allows developers to create workflows that execute automatically when specific events occur.

Examples:

- Push to repository
- Pull request creation
- Release creation
